* What This Thread Is About

How does a bare language model become a functional colleague?

Not through fine-tuning, not through tool access alone, but through a
progressive dressing of context that gives the spirit the awareness it
needs to act appropriately within an organisation. We call this
*situational awareness*: the layered knowledge that turns a substrate
into an agent.

This thread documents MāyāLucIA's empirical answer — a system that
has been running since February 2026 across multiple substrates,
hosts, and harnesses.
